# Hammerspoon Setup

Hammerspoon configuration with **Faces** - fast virtual workspaces, and **Spoonman** - a simple spoon management system.

## Installation

```bash
./install.sh
```

Or from the dotfiles root:

```bash
install/hammerspoon
```

This will:
1. Install Hammerspoon via Homebrew
2. Create `~/.spoons` directory
3. Symlink `upspoon` and `speeze` to `~/.local/bin`
4. Install Faces (symlink to `~/.hammerspoon/faces.lua`)
5. Create `~/.hammerspoon/init.lua`

## Faces

Faces = **Fa**st Spa**ces**

A minimal, high-performance virtual workspace manager. See [faces/README.md](faces/README.md) for details.

### Keybindings

| Keys | Action |
|------|--------|
| Ctrl+Space | Cycle to next face |
| Ctrl+Shift+Space | Cycle to previous face |
| Ctrl+Shift+1-6 | Switch to face 1-6 |
| Ctrl+Option+Shift+1-6 | Move window to face 1-6 |

## Spoon Management

### Directory Structure

```
~/.spoons/
├── SomeSpoon.spoon/          # git clone
└── spoonman                  # lockfile
```

### Commands

**upspoon** - Update spoons

```bash
upspoon                    # Update all spoons
upspoon SomeSpoon          # Update specific spoon
```

**speeze** - Freeze state to lockfile

```bash
speeze                     # Generate/update ~/.spoons/spoonman
```

### Lockfile Format

```
# spoonman - generated by speeze

SomeSpoon.spoon
  repo: https://github.com/someone/SomeSpoon.spoon.git
  sha: a1b2c3d4e5f6...
  branch: main
```

## Future Commands

Designed for but not yet implemented:
- `inspoon <repo-url>` - Clone a spoon
- `unspoon <name>` - Remove a spoon
- `respoon` - Restore from lockfile
